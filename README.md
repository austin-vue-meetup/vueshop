# vue-shopify

## Project setup

```sh
npm install
```

### Compiles and hot-reloads for development, runs api server at [http://localhost:3000](http://localhost:3000)

```sh
npm start
```

### Compiles and minifies for production

```sh
npm run build
```

### Lints and fixes files

```sh
npm run lint
```
